Deloitte Australia – Data Analytics Job Simulation
🔍 Overview

Completed the Deloitte Australia Data Analytics Virtual Job Simulation on Forage, solving a real-world manufacturing analytics case for Daikibo Industrials. The project focused on analyzing machine downtime data, identifying operational inefficiencies, and building an interactive Tableau dashboard to support business decision-making.

🧠 Key Objectives
Cleaned and analyzed manufacturing downtime data to identify equipment performance issues.
Built interactive Tableau dashboards to visualize downtime by device type and factory location.
Identified high-risk machines and factories experiencing the greatest operational downtime.
Generated actionable business insights to support maintenance planning and improve manufacturing efficiency.
Presented findings through a stakeholder-friendly dashboard for data-driven operational decision-making.

🛠️ Tools & Skills
Tableau – Interactive dashboards, charts, filters, and data visualization

Microsoft Excel – Data cleaning, preprocessing, and validation

Data Analysis – KPI analysis, downtime trend analysis, operational reporting

Business Intelligence – Dashboard design, storytelling, insight generation

Analytical Thinking – Root cause identification and performance optimization
